LaTeX Cheat Sheet

Basic Syntax

  • Inline: \( ... \) → Example: $a^2 + b^2 = c^2$
  • Display: \[ ... \] or equation environment

Functions

\sin, \cos, \tan, \log, \ln, \exp, \min, \max, \lim, \sup, \inf
\[
f(x) = \sin x + \log x
\]

Piecewise Functions

f(x) = \begin{cases}
x^2 & \text{if } x \geq 0 \\
-x & \text{otherwise}
\end{cases}

Summation

\sum_{i=1}^{n} i^2
\[
\sum_{i=0}^{\infty} \frac{1}{2^i}
\]

Integrals

\int f(x)\,dx
\int_{a}^{b} x^2\,dx
\iint, \iiint, \oint

Matrices

\begin{bmatrix}
a & b \\
c & d
\end{bmatrix}

Other styles: pmatrix, vmatrix, Vmatrix

Cases and Conditions

\begin{cases}
1 & \text{if } x > 0 \\
0 & \text{otherwise}
\end{cases}

Misc Tips

  • \,, \;, \! for spacing in math mode
  • \text{} for inserting regular text in math mode
x = 0 \text{ if } y > 2
